## 🍕 Task 05 — Food Calorie Estimation using Transfer Learning

**File:** `task5.py`

Built a food recognition and calorie estimation system using **MobileNetV2 Transfer Learning**.

This project classifies food images and retrieves nutritional information such as calories, protein, carbohydrates, and fats.

### Features

* Food Image Classification
* Nutrition Information Retrieval
* Calorie Estimation
* Real-world Food Prediction

### Concepts Used

* Transfer Learning
* MobileNetV2
* Computer Vision
* Deep Learning
* Nutrition Analysis

### Datasets

* Indian Food Classification Dataset (Kaggle)
* Custom Nutrition Dataset created manually for calorie estimation

### Note

Some food classes are occasionally misclassified when tested on real-world images. For example, certain Paani Puri images were predicted as Jalebi. This highlights the importance of dataset diversity and model generalization in real-world Machine Learning applications.
